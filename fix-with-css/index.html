Safety: Removing the Horizontal Scroll

  • Website: https://www.papersource.com/
  • CSS code: .page-wrapper { overflow: hidden; }
  • Safety: By removing the horizontal bar, I improved the Safety dimension of the website. The horizontal bar was not adding anything functionally to the website and it makes users confused on why it is even there. By using the trackpad to try and scroll left and right, it can accidentally activate the back button. This unintentionally takes the user back to the previous page, which can make them lose their place or progress.
